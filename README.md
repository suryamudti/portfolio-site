# 🚀 Premium Personal Portfolio Website

Welcome to the repository of **Suryamudti Salmat**'s personal portfolio! A state-of-the-art, high-performance web experience designed to showcase mobile development expertise, core projects, professional history, and credentials.

Built with a focus on modern web standards, sleek dark aesthetics, and seamless animations.

## ✨ Features

- 🌌 **Premium Dark Theme & Glassmorphism**: Tailored cohesive color palette featuring vibrant blue and violet accent gradients over a deep slate background.
- 📱 **Interactive Project Gallery**: Responsive grid containing direct links to key GitHub repositories like `Android-Live-Soccer`, `MyPokedex`, and `BookNote-iOS`.
- 💼 **Experience & Education Timeline**: Beautifully structured layout featuring comprehensive histories at **Grab**, **tiket.com**, and **PT Cranium Royal Aditama**, decorated with high-quality company branding.
- ⚡ **Animations & Smooth Transitions**: Interactive navbar scroll behaviors and staggered scroll-triggered fade-in animations for an immersive user experience.
- ⚡ **Ultra High Performance**: Pure HTML5, CSS3, and Vanilla JavaScript with zero external frameworks, providing instantaneous page loads and perfect SEO optimization.

## 🛠️ Core Technologies

- **Languages**: HTML5, CSS3, ES6+ JavaScript
- **Design Tokens**: Custom CSS Variables, Glassmorphism, CSS Transitions & Keyframes
- **APIs & Assets**: Vector SVGs, Official Brand Logos, Google Fonts (Outfit & Inter)

## 📂 Project Structure

## 🚀 Live Demo & Deployment

This site is optimized for one-click static hosting like **GitHub Pages**, **Vercel**, or **Netlify**.

To run locally:
1. Clone this repository: `git clone https://github.com/suryamudti/portfolio-site.git`
2. Open `index.html` in any web browser.
